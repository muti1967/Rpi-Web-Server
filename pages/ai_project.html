AI Emotion Detector

Skills Used: Python, YOLO, OpenCV, AI, AWS, Google Colab, Labelbox, Model Training

In this project, I collaborated with my team to develop AIEIO, an AI-driven emotion detection system using the YOLO neural network. We focused on creating a robust dataset by scripting tools for data collection, labeling, and formatting to meet YOLO's input requirements. My contributions included implementing scripts to collect images via APIs and OpenCV, organizing data with tools like Labelbox, and automating the conversion of labels into YOLO-compatible formats. Additionally, I supported model training, evaluation, and deployment across platforms like Google Colab and AWS, ensuring efficient usage of GPU resources and performance tuning.
