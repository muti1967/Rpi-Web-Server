AI Gesture Recognition Device

Skills Used: IOT, Swift, TensorFlow, Python, C++, Soldering, CAD

My team and I are currently developing a gesture recognition device that will help children and adults with Autism manage their day to day tasks. We developed a wearable, gesture-controlled reminder device to assist people with special needs with task management and reduce harmful repetitive behaviors. The device integrates IoT, AI, and ML technologies, utilizing a Raspberry Pi Zero 2 W, PAJ7620U2 Gesture Sensor, and custom software applications built in C++ and Swift for task management and real-time synchronization via Bluetooth. Key features include gesture recognition, task reminders, calming message output, and portability with a compact, 3D-printed housing. The project focused on affordability, accessibility, and privacy, incorporating user feedback from initial testing to enhance functionality and usability. This project is still in development and we are currently working on improving the device and adding more features! Please reach out if you would like to learn more.
